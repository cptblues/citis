# Documents de conception

Ces documents sont les références initiales du projet.

Le nom actuel du jeu est **Citis**. Les occurrences de « Renaissance » ou « Virea » dans les documents correspondent aux anciens noms de travail.

Pour le premier prototype, l'ordre de priorité est :

1. carte fictive claire et lisible ;
2. placement de structures ;
3. influences entre hexagones ;
4. transformations visuelles satisfaisantes ;
5. score et synergies ;
6. communes réelles dans une étape ultérieure.
